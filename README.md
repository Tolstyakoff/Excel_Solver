# Excel_Solver
задачи на поиск решения Excel
5 задач.
1. Оптимальный план производства
   
- Предположим, что в трех цехах (Ц1, Ц2,Ц3) изготавливаются два вида из-делий И1 и И2. Известна загрузка каждого цеха Aij (оцениваемая в данном случае в процентах)
при изготовлении каждого из изделий и цена реализуе-мой продукции в рублях сi.(за1 изделие).
- Определить, сколько изделий каждого вида следует производить при возможно более полной загрузке цехов, чтобы получить за рассмат-риваемый плановый период максимальную прибыль.
