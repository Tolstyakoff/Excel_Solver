**5 задач.**
**1. Оптимальный план производства**
Решить следующую задачу с помощью Поиска решения MS Excel. 
Предположим, что в трех цехах (Ц1, Ц2,Ц3) изготавливаются два вида изделий И1 и И2. Известна загрузка каждого цеха  
аij (оцениваемая в данном случае в процентах) при изготовлении каждого из изделий и цена реализуемой продукции в рублях 
сi.(за1 изделие) - данные в таблице А

Определить, сколько изделий каждого вида следует производить при возможно более полной загрузке цехов, чтобы получить за рассматриваемый плановый период максимальную прибыль.
Решить задачу также для значений 
с1 из таблицы B
предполагая во всех случаях, что 
с2. остается неизменной 
В каком случае оптимальный план включает только один вид изделий?

и т.д. формулировки задач размещены в листах Excel
